## _config.yml设置

~~~
#markdown解释器设置为kramdown, 因为github对默认的Maruku支持不好

#使用disqus评论
#设置disqus帐号. 
#disqus默认有广告(dicovery box), 在 http://USERNAME.disqus.com/admin/settings/页面，将"discovery :show links to ...."去掉勾选 
      short_name : roothoo 
  comments :
    provider : disqus
    disqus : 
      short_name : roothoo 
~~~
