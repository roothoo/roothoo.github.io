1. 粗体  
\*\*\*word\*\*\*  
例子:  
**username**  


2. 代码  
\`\`\`  
code  
\```  
例子：  
```
for (i = 0; i < 10; i++)
	printf ("%d\n", i);
```  

3. 换行  
空格 空格 回车  
例子:  
上一行  
下一行
同一行  

4. 强调  
\`word\`  
例子：  
`word`

\* blue
\* yellow
